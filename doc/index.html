hash

Generate a base64-encoded SHA-256 hash for a given data.
Objects can be passed to this function but will generate the same hash regardless of its content.

hash
Parameters
data ((string | number | Array | Date)) Data to encode
Returns
string: Base64 encoded SHA-256 hash
Example
hash('Lorem Ipsum dolore sit amet'); //returns 'sha256-7almix3trlcKWVAN+fhV/Bzbx4BixTwzjYpZDWUxctM='

writeToFile

Write to a file (while prettifying the content).

writeToFile
Parameters
filename (string) File name
data (Array<string>) Lines to write to the file
outputFormat (string = OUTPUT_FORMAT) Format of the output
Throws
  • Error: No filename specified
  • IoError: Writing error
Example
writeToFile('output.txt', ['Lorem ipsum dolore sit amet']);

With a specified format:

writeToFile('output.json', [{key: 'val'}], 'json');
writeToFile('output.csv', ['0,john,doe', '1,lorem,ipsum'], 'csv');

prettifyOutput

Prettify the ouptut according to the format used or keep it as it is.

prettifyOutput
Parameters
output ((string | Object | Array<string>)) Output
format (string = OUTPUT_FORMAT) Format of the output (json, csv, ...)
Returns
string: Prettified output
Example
prettifyOutput('some unchanged text'); //returns 'some unchanged text'

With a specified format:

prettifyOutput({key: 'val'}, 'json'); //returns {<br>  "key": "val"<br>}
prettifyOutput('0,john,doe', 'csv'); //returns '0, joh, doe'

scanInput

Scan an input and output it's integrity hash.

scanInput
Parameters
input ((string | Array<string>)) Input to hash (e.g. JS code)
noOutput (boolean = false) Don't output the result to the terminal but return the hash
Returns
(void | string): Hashed data or nothing
Throws
  • Error: Hashing or input error
Example
scanInput('Lorem ipsum dolore sit amet'); //logs 'sha256-HBQ/am1i8gw1bl8qJDhm0naAsChqeYsBEiCWTRLEaE8='
scanInput(['Lorem ipsum dolore sit amet']); //logs the same as above

Without output disabled

scanInput('Lorem ipsum dolore sit amet', true); //returns 'sha256-HBQ/am1i8gw1bl8qJDhm0naAsChqeYsBEiCWTRLEaE8='
scanInput(['Lorem ipsum dolore sit amet']); //returns the same as above

readFilesSync

Synchronously read files and scan them.

readFilesSync
Parameters
files (Array<string>? = process.argv.slice(2,process.argv.length)) Array of file paths
obj (Config = {}) Configuration
Name Description
obj.prettify any (default false)
obj.outputDest any (default OUTPUT_DEST)
obj.outputFormat any (default OUTPUT_FORMAT)
Example

Reading from the CLI

readFilesSync();

Reading from specific files

readFilesSync(['output.txt']);

... With specific configurations

readFilesSync(['input.json'], {prettify: true, outputFormat: 'json'}); //logs {<br>  "output.json": "sha256-iTyF6rE+vAUIIWrWaC6bWt9NwI/74kpOuk4JZl9zCMM="<br>}
readFilesSync(['input.csv'], {outputDest: 'output.json', outputFormat: 'json'}); //Writes the above to output.json

run

Start the hasher.

run
Parameters
files (Array<string> = []) List of files to go through
obj (Object = {}) Configuration
Name Description
obj.format string (default 'text') Format of the output (text, json, csv)
obj.input string (default 'any') Location of the input (any, stdin, args)
obj.output string (default 'stdout') Destination for the output (stdout, var, filenames )
obj.prettify boolean (default false) Prettify the output
Returns
(undefined | Array<string> | string): Data or nothing
Example

Reading from files or STDIN

run();

Reading from files

run(['index.js']);
run(['index.js'], {input: 'args'});

Reading from STDIN

run([]);
run([], {input: 'stdin'});

Reading from files with configurations

run(['index.js', 'index.css'], {format: 'json', output: 'index-hash.json', prettify: true});

Reading from STDIN with configurations

run([], {format: 'csv', input: 'stdin', prettify: true});

EOF

Check if a string is deemed to be an EOF symbol (\$ or \EOF).

EOF
Parameters
str (string) Text
Returns
boolean: Is it an EOF character?
Example
EOF('\n'); //returns false
EOF('END\$'); //returns false
EOF('\\$'); //returns true
EOF('\\EOF'); //returns true
EOF('\$'); //returns false
EOF('\EOF'); //returns false
